# Simple CRUD TS

## Descripción
* Programa para hacer un CRUD simple,con TypeScript, usando NodeJS y Handlebars

## Tecnologías
* NodeJS
* Express
* Express-Handlebars
* Handlebars
* Helmet
* Cors
* Morgan
* Rimraf
* Compression
* TypeScript

## Ejecución
* Importar módulos con npm install
* npm run start, para ejecutar el programa
* Pobrar el Server en http://localhost:3000/
